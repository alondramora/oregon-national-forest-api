Oregon National Forest API

How to use

To use this api, please use the following url: https://oregon-national-forest-api.herokuapp.com/api/forestName

Example 1: https://oregon-national-forest-api.herokuapp.com/api/umpqua
Example 2: https://oregon-national-forest-api.herokuapp.com/api/fremontwinema
