Oregon National Forests API

How to use

To use this api, please use the following url: http://localhost:8000/api/forestName

Example 1: http://localhost:8000/api/umpqua
Example 2: http://localhost:8000/api/fremontwinema

Forests

  • Fremont-Winema National Forest
  • Malheur National Forest
